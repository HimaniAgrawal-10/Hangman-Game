# 🎯 Hangman Game

An interactive and fun Hangman game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* 🎨 Animated hangman drawing (Canvas)
* ⌨️ A-Z interactive keyboard + typing support
* ❤️ Lives counter
* 🏆 Score system
* 📂 Category-based words
* 🌗 Dark mode toggle
* 🔄 Restart game

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)
